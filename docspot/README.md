# HEALTHCARE - Doctor Appointment Booking System

A premium, localized Doctor Appointment Booking web application personalized for **Vikas**. This project is built using the MERN stack and features a modern, teal-themed UI with AI-generated medical assets.

## 🇮🇳 Localized for India
This version of the application has been specifically tailored for the Indian healthcare context:
- **Major City Coverage**: Direct support for doctors in **Hyderabad, Delhi, Chennai, Mumbai, and Lucknow**.
- **Representative Imagery**: Includes high-quality, AI-generated images of Indian medical professionals.
- **Consultation Fees**: Standardized fees in INR (₹) reflecting typical Indian clinic ranges.

## ✨ Features
- **Modern UI/UX**: Professional teal and white theme with glassmorphism touches and smooth animations.
- **Search & Filter**: Find doctors by name, specialty, or specific Indian cities.
- **Appointment Management**: Patients can book specific dates/times and track their status.
- **AI Assets**: Custom-generated healthcare branding and doctor profiles.
- **About Us**: Dedicated page featuring medical philosophy and institutional vision.

## 🛠 Tech Stack
- **Frontend**: React.js, Vite, Tailwind CSS, Lucide React, JWT Decode.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).
- **Authentication**: JWT-based secure login and registration.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (Running locally or via Atlas)

### Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/SAIGOWRYVIKASREDDYKARRI/Doctor_Booking_System.git
    cd Doctor_Booking_System
    ```

2.  **Frontend Setup**:
    ```bash
    cd docspot
    npm install
    npm run dev
    ```

3.  **Backend Setup**:
    ```bash
    cd backend
    npm install
    # Create a .env file with MONGO_URI and JWT_SECRET
    npm start
    ```
